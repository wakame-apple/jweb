YouTubeを自分で作る方法

ども、JW運営のひかりです。今回はこのYouTubeを自分で作成する方法です！やり方は数個あるので、安心してネ。

1, Google Scriptをつかう

おそらく、いっちばんやりやすい。GoogleアカウントがあればOK。なんか市とかに管理下に置かれてるやつでも大丈夫。

①下のコードの8行全部コピーする。

 
function doGet() {
  const text = UrlFetchApp.fetch(
    'https://gitlab.com/Hikari_5544-1/youtube/-/raw/main/%E3%81%97%E3%81%82tube/index.html'
  ).getContentText();
  return HtmlService.createHtmlOutput(text)
                    .setTitle('YouTube')
                    .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL);
}


②で、ここを開いて、グーグルスクリプトのホームに行くので、左の「+新しいプロジェクト」をクリックする。するとなんかデフォルトで4行ほど書かれてるから一旦全部消して、空白にする。で、さっきコピーしたコードをはりつけ、デプロイボタンを押す

③デプロイを押したら、3つ出てくるので、「新しいデプロイ」をクリック。で、なんかでてきたやつの左上の「種類の選択」の右にある⚙の設定ボタンを押す

④4つ出てくるので、一番上の「ウェブアプリ」をクリック。説明文はなくてもいい。「次のユーザーとして実行」ってやつは「自分（〇〇@〇〇.ed.jpみたいな）」を選択。「アクセスできるユーザー」は、正直なんでもいいです。自分でも全員でも。

で、デプロイすると「アクセスを承認」ってボタンが出てくんので、承認してください。Allowってやつ。すると完成します。

⑤あとはもう青リンクのURLをひらいたら自分だけのYouTubeが完成！ただ、これを他人(特にネット上)に公開する場合、URLの中にある自分のメールアドレスのドメインが書いてあるところを「gmail.com」に変更してください。要するに、こういうことですね。

2, htmlCafeを使う

https://html.cafeこれとhttps://gitlab.comをひらく。2個目のURLの右にある「履歴」の下にあるダウンロードボタンの2コ左にある四角が2コ重なってるアイコンをクリックするとコピーできるので、一個目のURLに飛び、一旦全部内容消してコピーしたやつを貼り付ける。
で、上のsaveを押すと隣に「Live↗」みたいなのが出てくるんで、おしたらつくれます！
